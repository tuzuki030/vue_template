# vue_template

テンプレートになるかは微妙ですが、以下の構成で動作するサンプルです。

- vue.js
- CSSフレームワーク
  - Bulma(https://bulma.io)
- HTTPクライアント
  - axios(https://github.com/axios/axios)
- テストフレームワーク
  - jest(https://jestjs.io)


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
